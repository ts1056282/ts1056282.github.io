Verificação de antigas URLS ainda indexadas nos resultados do Google, que geram erros 404

Mensuração dos erros 404

Quando se migra um site, as URLs do site antigo continuam sendo mostradas nos resultados do Google. O usuários que clica nessas, acaba vendo uma página de erro, isto penaliza o site, fazendo com que este perca bons posicionamentos no Google relativos a outras palavras-chave específicas. Na imagem ao lado, um exemplo de como as URLs que geram erro 404 são reportadas, via plataforma com interface gráfica.

Redirecionamento das páginas de erro

Para sanar o problema das URLs que geram erro 404 para os usuários, é preciso que estas sejam redirecionadas para as novas páginas. Na imagem ao lado, um exemplo de como o redirecionamento é feito, via plataforma com interface gráfica.

Remoção das páginas de erro

Algumas URLs não precisam de ser redirecionadas, apenas removidas dos resultados do Google, pois são referentes a páginas ou arquivos não mais existentes no novo site. Pra isso, são feitos pedidos de remoção de URLs diretamente no Google Webmaster Tools.
